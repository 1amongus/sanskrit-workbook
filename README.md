# संस्कृत स्वाध्याय — Sanskrit Workbook

An interactive HTML workbook for completing exercises from **Samskrita Swadhyaya Tritiya Diksha** (Saṅkṣēpa Rāmāyaṇa) published by Rashtriya Sanskrit Sansthan.

## 🌐 Live Demo

**[Open Workbook →](https://1amongus.github.io/sanskrit-workbook/)**

## ✨ Features

- **74 exercises** organized as chapters (अभ्यासाः)
- **Sidebar navigation** with chapter list
- **Progress tracking** — saved to browser localStorage
- **Notes** — add personal notes per exercise
- **Image-based exercises** — original scanned exercises displayed inline
- **No server required** — works entirely in the browser

## 🚀 Getting Started

### Online
Visit: https://1amongus.github.io/sanskrit-workbook/

### Offline
1. Clone: `git clone https://github.com/1amongus/sanskrit-workbook.git`
2. Open `index.html` in any browser

## 📁 Structure

```
├── index.html              # Main workbook app
├── exercises_data.js       # Exercise metadata
├── exercise_manifest.json  # Image mapping
├── exercise_images/        # 74 scanned exercise images (PNG)
└── *.pdf                   # Source PDF
```
